# DriveThruRPG SDK

Meta-repository to organize DTRPG SDK libraries and documentation

## About This Project

This project creates an SDK for a given language that can be used for a client application (CLI or GUI)
to interact with the DriveThruRPG API.

## Submodules

- `api` - contains API documentation that all other submodules use to generate code based on the API
- `go` - Go language version of the SDK
- `python` - Python language version of the SDK
- `rust` - Rust language version of the SDK
- `swift` - Swift language version of the SDK
